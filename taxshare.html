TaxiShare Mobile
App Concept
PROJECT OVERVIEW
TaxiShare is a mobile app concept that I worked on as a personal project. Ever thought cab fare was too pricey? Me too, therefore this app helps taxi riders find other riders nearby, and match them up if they are headed in the same direction to share the cost. For the design, I wanted to show where the user is on the map, and it only need 2 questions from the user to generate matches. Once generated, the matches have all the details to where and when the user is leaving, and they can message the person if they want to share a cab.


ROLES Ideation, concepting, wireframing researching, UI/UX design, graphic design

TOOLS Adobe Illustrator, Adobe XD, CSS/SASS
